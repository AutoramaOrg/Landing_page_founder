# Glossário

## Autorama Racing

Nome do produto/jogo apresentado pela landing.

## Pacotes de Fundador

Oferta principal da página. Divide os benefícios em três tiers: Bronze, Prata e Ouro.

## Hero

Primeira área da página, visível no topo. Atualmente usa `public/hero-celebration.png` como imagem de fundo full-bleed atrás dos textos.

## Full-bleed

Imagem ou seção que ocupa a largura inteira da viewport, sem ficar presa a um card ou quadro lateral.

## CTA

Call to action. Botão ou link usado para conduzir o visitante para uma ação. Nesta landing, os CTAs atuais apontam para âncoras internas.

## Build

Processo que gera os arquivos finais em `dist/` usando `npm run build`.

## Preview

Servidor local para visualizar a build final via `npm run preview`.

## Asset público

Arquivo em `public/` servido diretamente pela aplicação. Exemplo: `/autorama_white.png` e `/hero-celebration.png`.

## Arquivo gerado

`src/App.js` é gerado pelo script `scripts/compile-jsx.mjs` a partir de `src/App.jsx`.
