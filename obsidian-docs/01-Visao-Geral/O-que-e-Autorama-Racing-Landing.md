# O que é a landing Autorama Racing

## Resumo

A landing page do **Autorama Racing** apresenta os **Pacotes de Fundador** do jogo. A página comunica acesso antecipado, itens exclusivos e status de fundador, conduzindo o visitante para a seção de pacotes.

## Escopo confirmado

- Landing page estática/client-side.
- Frontend em React 18.
- Build com Vite.
- Estilização com Tailwind CSS e CSS customizado.
- Conteúdo principal centralizado em `src/App.jsx`.
- CSS principal em `src/index.css`.
- Assets públicos em `public/`.

## Escopo não confirmado

- Checkout real.
- Integração com gateway de pagamento.
- Captura de lead.
- Analytics.
- Deploy em produção.
- Domínio final.

Esses itens devem ser tratados como `A confirmar` até existirem artefatos reais no repositório ou no ambiente de produção.

## Proposta da página

A página vende uma experiência de entrada antecipada no universo Autorama Racing:

- acesso antecipado ao jogo;
- itens e badges de fundador;
- pacotes Bronze, Prata e Ouro;
- reforço de exclusividade e urgência.

## Arquivos centrais

- `src/App.jsx`: conteúdo, dados das seções e componentes React.
- `src/App.js`: arquivo gerado a partir de `src/App.jsx` pelo script de compilação.
- `src/index.css`: sistema visual, responsividade, hero e componentes.
- `src/main.js`: bootstrap React usando `ReactDOM` e `React` globais.
- `index.html`: HTML base e carregamento dos vendors públicos.
- `public/autorama_white.png`: logo branco atual.
- `public/hero-celebration.png`: imagem full-bleed do topo.
